# 分类
在具体分类下需要在顶部合适的位置展示分类类别。如“主分类”、“子分类”。
“主分类”下，点击“新增”时，直接调整到新增主分类页面，“标题”必填。
“子分类”下，顶部有“图片”、“文本”tab，默认图片。在什么tab下，点击新增跳转到对应的创建页面。
